# Excel Sales Dashboard Project

## 📊 Project Overview

This project was created entirely in Microsoft Excel to analyze sales data and generate meaningful business insights. The dashboard helps track sales performance, identify trends, and support data-driven decision-making.

## 🎯 Objectives

* Analyze sales performance across different regions and products.
* Monitor key performance indicators (KPIs).
* Identify top-performing products and sales trends.
* Create an interactive and user-friendly dashboard.

## 🛠 Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Conditional Formatting
* Slicers
* Excel Formulas (SUMIF, COUNTIF, XLOOKUP, IF, INDEX-MATCH, etc.)

## 📈 Dashboard Features

* Total Sales KPI
* Total Orders KPI
* Region-wise Sales Analysis
* Product-wise Performance
* Monthly Sales Trends
* Interactive Filters using Slicers

## 🔍 Data Analysis Performed

* Data Cleaning
* Duplicate Removal
* Missing Value Handling
* Data Validation
* Trend Analysis
* Performance Reporting

## 📁 Files Included

* Excel Dashboard (.xlsx)
* Raw Dataset
* Dashboard Screenshot(s)

## 💡 Key Insights

* Identified top-performing regions and products.
* Analyzed monthly sales growth patterns.
* Highlighted areas requiring business attention.
* Improved reporting efficiency through dashboard automation.
